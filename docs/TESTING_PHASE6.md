## Common Issues

### Issue: "yosys: command not found"

**Solution**: 
```bash
cd /home/nixos/repos/clash-vscode-yosys
nix develop
code .
```

### Issue: Synthesis fails with "hierarchy" error

**Cause**: Top module name mismatch

**Debug**:
```bash
# Check what module name Clash generated
grep "^module " test-project/.clash-synth/verilog/**/*.v
```

**Fix**: Ensure using correct module name (lowercase with underscores)

### Issue: Statistics not showing

**Possible Causes**:
1. Yosys version too old (need recent version)
2. `stat` command format changed

**Debug**: Check raw output in output channel

### Issue: JSON file not created

**Cause**: `write_json` might not be supported in Yosys version

**Solution**: Update Yosys or disable JSON output
